Learning Goals

Windows Server 2022 is the latest version of the Microsoft Server Operating System. Building on the foundation of previous Windows Server editions, Windows Server 2022 enhances the technologies you already know and use. Especially in the areas of security, containers and integration with the cloud important improvements were made.

JEA: Just Enough Administration

Just Enough Administration (JEA) is a security technology that enables delegated administration for anything that can be managed with PowerShell. JEA allows you to reduce the number of administrators on your machines by leveraging virtual accounts or group managed service accounts that perform privileged actions on behalf of regular users. It limits what users can do by specifying which cmdlets, functions, and external commands they can run.

Windows Server Containers

A container is a standard unit of software that packages code and all its dependencies so the application runs quickly and reliably from one computing environment to another. A Docker container image is a lightweight, standalone, executable package of software that includes everything needed to run an application: code, runtime, system tools, system libraries and settings.
