Learning Goals

Microsoft Copilot Studio allows you to create agents or bots through a no-code graphical interface without having to be an AI developer or data scientist. Microsoft Copilot Studio lets you quickly create and publish agents to Microsoft 365 or any external site based on your own unique business needs. This course gets you up and running with Microsoft Copilot Studio so you can quickly start building your own agents.

Target audience

This 3-day course is intended for people who want to build their own agent with Microsoft Copilot Studio. No prior knowledge is required.

Building Autonomous Agents

Autonomous agents can automatically respond to signals across your business and initiate tasks. They are configured to automatically react to events or triggers without human input that instead originate from various tools, systems, and databases, or are even scheduled to run hourly, daily, weekly, or monthly. Autonomous agents enable customers to move from legacy line of business applications to AI-first business processes.

Publishing and Monitoring an Agent

Once your agent has been developed and tested, you can publish it to engage with your customers on multiple platforms or channels. These include Microsoft 365, Microsoft Teams, websites, mobile apps, Facebook...
