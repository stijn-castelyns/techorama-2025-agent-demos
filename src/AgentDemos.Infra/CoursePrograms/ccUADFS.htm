Learning Goals

This four-day course provides students with the knowledge and skills to install and configure Active Directory Federation Services. The course focuses on terminology, user interfaces and common configuration scenarios for AD FS. Participants will learn how to design AD FS environments and supporting technology such as a PKI Infrastructure. They will also learn how to design AD FS for security and high availability. This training focuses on AD FS 4.0 and AD FS 5.0.

Target Audience

This course is intended for IT administrators who are responsible for the setup and configuration of claims-based authentication, whether on premises or to connect to services in the cloud.
