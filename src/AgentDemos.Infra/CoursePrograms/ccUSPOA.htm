Learning Goals

This course aims to gives students a full overview of all capabilities that the SharePoint Premium license adds to SharePoint Online.

Target Audience

This course is intended for users that want to now the extended capabilities that the SharePoint Premium license has to offer. This course targets both Power Users and Administrators for SharePoint Online.

Prerequisites

This course dives deeper into SharePoint, so basic knowledge about SharePoint Online is recommended.
