> AI Assisted Development with GitHub Copilot

Learning Goals

Generative AI models like GPT-4o from OpenAI are becoming better and better at writing code. Modern developers who embrace this new technology, can significantly speed up their development tasks, going from implementing new features to writing documentation and unit tests. This course shows how developers can harness the power of GenAI right from their IDE by using GitHub Copilot.

Target Audience

This course is meant for developers looking to increase their productivity using Generative AI through GitHub Copilot. All labs and demos are based on C# code, but the principles covered can be applied to any programming language.

Course Outline

Introduction to GitHub Copilot

Discover the fundamentals of AI-powered development with GitHub Copilot. Learn how this revolutionary tool enhances developer workflow through intelligent code suggestions while understanding its capabilities and limitations.

  • What is AI pair programming?
  • GitHub Copilot Licenses
  • Setting up Copilot in your environment
  • Understanding privacy and security considerations
  • LAB: Activating GitHub Copilot Free License

Using GitHub Copilot

Find out how to get the most out of GitHub Copilot as a developer by discovering all of its features and how it integrates into both your IDE and source control system.

  • Code Completions
  • Copilot Chat
  • Git Integration
  • Debugging Assistance
  • Copilot for CLI Commands
  • LAB: Implement a new feature using Copilot

GitHub Copilot Prompt Engineering and Model Selection

Learn how to craft effective prompts and contextual cues to increase Copilot's code generation capabilities. Explore best practices for guiding AI pair programming by giving clear commands dependent on the AI model you are working with.

  • What is Prompt Engineering?
  • Model Selection: Knowledge vs Reasoning Models
  • Few-Shot Prompting
  • Iterative prompting strategies
  • Structuring Prompts with Markdown and XML
  • LAB: Guiding Copilot with Prompt Engineering

Making GitHub Copilot Smarter with Extensions

Enhance Copilot's capabilities by integrating custom extensions and third-party tools to query documentation, use third-party AI models and communicate with external systems. Discover how to tailor Copilot to your specific tech stack and workflow requirements.

  • Using Language/framework-specific extensions
  • Creating a GitHub Copilot Skillset
  • Building a GitHub Copilot Agent
  • LAB: Building an extension for querying documentation

Implementing Entire Features with GitHub Copilot Workspace

GitHub Copilot Workspace allows you to let copilot autonomously implement entire features, starting from high-level requirements. Through an iterative back-and-forth you can steer copilot precisely in the direction you want, without even having to use an IDE!

  • Introduction to GitHub Copilot Workspace
  • Defining Feature Requirements with GitHub Issues
  • Steering Copilot
  • Reviewing Changes with GitHub Codespaces
